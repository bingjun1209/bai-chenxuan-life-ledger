# 白宸瑄｜我的生活記帳

這是一個結合個人介紹與日常記帳功能的單頁網站。

## 功能

- 新增與刪除消費紀錄
- 本月支出、今日支出與紀錄筆數統計
- 依關鍵字、類別與月份篩選
- 類別支出比例
- 瀏覽器本機保存資料（LocalStorage）
- 匯出 CSV
- 手機與電腦版面自動調整

## 開啟方式

直接雙擊 `index.html` 即可在瀏覽器中開啟。

## 發布到 GitHub Pages

1. 在 GitHub 建立一個新的 Repository。
2. 上傳本資料夾內的所有檔案。
3. 進入 Repository 的 `Settings`。
4. 左側選擇 `Pages`。
5. 在 `Build and deployment` 中選擇 `Deploy from a branch`。
6. Branch 選擇 `main`，資料夾選擇 `/root`，按下 `Save`。
7. 等待幾分鐘後即可取得公開網址。
